## soundpicker-server

짧은 시간 노래듣고 제목 맞추기 게임 [사운드피커](https://soundpicker.kr) 웹서비스의 서버입니다.

### Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
